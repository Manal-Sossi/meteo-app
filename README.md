# 🌤️ Météo App
Application web de météo en temps réel développée en JavaScript vanilla.

## Fonctionnalités
- Recherche météo par ville
- Température, humidité, vitesse du vent
- Historique des 5 dernières villes (LocalStorage)

## Technologies utilisées
- JavaScript (Fetch API, Async/Await)
- HTML / CSS
- OpenWeatherMap API
- LocalStorage

## Comment utiliser
1. Ouvrir `index_jour14_Mini-projet_Météo_App.html` dans un navigateur
2. Entrer une ville dans le champ de recherche
3. Appuyer sur Entrée ou cliquer sur Rechercher
